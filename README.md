# Android-Awsome-Onboarding
Awsome android motion onboarding screen

### Features:

    - animation
    - onboarding
    - google+ sign 
    - Dashboard

### Application Uses:

    Database                  : Firebase
    Login and Authentication  : google+
    Language                  : Java
    Tools                     : Android Studio
